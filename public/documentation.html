This is an API that will distribute the information about movies I've seen.

Request URL HTTP Method Request body data format Response body data format
Welcome Screen for myFlix / GET None Text with a Welcome message
Get all movies by title /movies GET None A JSON holding a list of each movie title
Get movie details /movies/[title] GET None A JSON holding details about a movie containing name, movie description, related genres, director, and image. Example:
{
title: 'Joker',

description:'Crazy clown likes to laugh creepy and kill people.',

genre:{
name: 'crime'
description:'people doing bad things'
}

director: {
name: 'Todd Phillips',
bio: 'American director and screenwriter'
birth: '1965'
}

image: 'joker.png',

actors: ['Jaoquin Phoenix', 'Robert DeNiro']
}
Get all genres by title /genres GET None A JSON holding a list of each movie genre
Get genre details /genres/[genre_name] GET None A JSON holding details about a movie genre containing name, and genre description. Example:
{
genre_name: 'thriller',
description:"scary stuff",
}
Get a list of directors by name /directors GET None A JSON holding a list of each movie director
Get director details /directors/[director_name] GET None A JSON holding details about a movie director containing name, movies directed, and birth date. Example:
{
director_name: 'Jon Favreau',
movies_directed: 'Iron Man',
birth_date: 'October 19, 1996'
}
Create a new user account /Account POST A JSON with the name, username, password, email, and dob structured like:
{
name: 'Jake Kirby',
username: 'example-username',
password: '1234'
email: 'kirby_jake@aol.com'
dob: 'July 12, 1919'
} 		A JSON object holding data about the user that was added:
{
name: 'Jake Kirby',
username: 'example-username',
password: '1234'
email: 'kirby_jake@aol.com'
dob: 'July 12, 1919'
}
Get account details /Account/[username] GET None A JSON holding details about a user account containing name, username, password, email, and dob structured like:
{
name: 'Jake Kirby',
username: 'example-username',
password: '1234'
email: 'kirby_jake@aol.com'
dob: 'July 12, 1919'
}
Delete user account /Account/[username] DELETE None A text message stating that the user account was deleted.
Edit user account /Account/[username]/edit PUT None A text message stating that the user account was updated.
Add movie to favorites /Account/favorites/[title] POST None A text message stating that the movie was added to favorites.
Remove movie from favorites /Account/favorites/[title] DELETE None A text message stating that the movie was removed from favorites.
